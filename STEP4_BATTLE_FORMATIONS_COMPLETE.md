## 🎮 Три режима построений

### 1️⃣ **BridgeFormation** (Боевой мост - 1 на 1)

```
Структура:
┌─────────────┐       ┌─────────────┐
│ На ринге    │       │ На ринге    │
│ Боец 0 🔥   │ атак  │ Боец 0 🔥   │
└─────────────┘       └─────────────┘
┌─────────────┐       ┌─────────────┐
│ Запас:      │       │ Запас:      │
│ Боец 1 ⏳   │       │ Боец 1 ⏳   │
│ Боец 2 ⏳   │       │ Боец 2 ⏳   │
└─────────────┘       └─────────────┘
```

- **На ринге**: только индекс 0
- **В запасе**: индексы 1+
- **Смерть**: следующий автоматически займет место (FirstOrDefault)
- **Соседи**: нет (каждый один)
- **Рендж**: отсутствует

**Код:**
```csharp
// Текущий юнит - первый живой
GetRingUnits() → Units.FirstOrDefault(u => u.IsAlive)
```

### 2️⃣ **ThreeOnThreeFormation** (3 на 3)

```
Структура 3 колонки:

На ринге:          Запас 1:        Запас 2:
1 [0,0] 💪      4 [1,0] ⏳       7 [2,0] ⏳
2 [0,1] 💪      5 [1,1] ⏳       8 [2,1] ⏳
3 [0,2] 💪      6 [1,2] ⏳       9 [2,2] ⏳

Index → (row, col):
0 → (0, 0)
1 → (0, 1)
2 → (0, 2)
3 → (1, 0)  ← следующий ряд
4 → (1, 1)
...

Позиция:
row = индекс / 3
col = индекс % 3
```

- **На ринге**: row == 0 (3 юнита)
- **В запасе**: row > 0 (9+ юнитов)
- **Противники**: одна колонка противнику (col совпадает)
- **Смерть**: юнит удаляется, остальные пересчитываются автоматически
- **Соседи**: 8 соседей + диагонали
- **Рендж**: Chebyshev distance (максимум разницы по осям)

**Примеры соседства:**
```
Юнит на [0,1] (индекс 1) имеет соседей:
- [0,0] (индекс 0) - слева
- [0,2] (индекс 2) - справа
- [1,0] (индекс 3) - снизу-слева
- [1,1] (индекс 4) - снизу
- [1,2] (индекс 5) - снизу-справа
(и диагонали в верхнем ряду, если существуют)
```

**Код:**
```csharp
// Получить позицию
int row = index / 3;
int col = index % 3;

// Проверить соседство (8-way distance)
Math.Abs(row1 - row2) <= 1 && Math.Abs(col1 - col2) <= 1
```

### 3️⃣ **WallToWallFormation** (Стенка на стенку)

```
Структура - вертикальный ряд:

Армия 1             Армия 2
┌──────┐            ┌──────┐
│1 💪  │ ───────── │A 💪  │ индекс 0
│2 💪  │ ───────── │B 💪  │ индекс 1
│3 💪  │ ───────── │C 💪  │ индекс 2
│4 💪  │ ───────── │D 💪  │ индекс 3
│5 💪  │ ───────── │E 💪  │ индекс 4
│6 ⏳  │           │       │ индекс 5 (в запасе - нет пары)
└──────┘            └──────┘

Позиция:
row = индекс
col = 0 (всегда единственная колонна)
```

- **На ринге**: min(army1.Count, army2.Count) юнитов
- **В запасе**: без пары (если одна армия больше)
- **Противники**: юнит с тем же индексом
- **Смерть**: юнит удаляется, остальные сдвигаются вверх (схлопываются)
- **БЕЗ ДЫР**: на ринге всегда целый фронт!
- **Соседи**: сверху/снизу по индексу (диагонали не применяются)
- **Рендж**: расстояние = разница индексов

**Пример смерти:**
```
ДО:  Индексы 0,1,2,3,4,5 живы
СМЕРТЬ индекса 3
ПОСЛЕ: Индексы переэумеруются как 0,1,2,3,4
       → Юниты "схлопываются" вверх, дыр нет!
```

**Код:**
```csharp
// На ринге
GetRingUnits() → первые N живых юнитов

// Противник по индексу
opponentArmy.Units[myIndex]

// Соседи - выше/ниже
neighbors = myIndex-1, myIndex+1
```

## 🎯 Ключевые особенности

### Автоматические замены (HandleUnitDeath)

| Режим | Алгоритм замены |
|-------|-----------------|
| **Bridge** | Следующий автоматически (FirstOrDefault) |
| **3на3** | Пересчёт индексов, поднялась замена автоматически |
| **Стенка** | Сдвиг вверх (схлопывание), дыр нет |

### Рендж спецсособностей

| Режим | GetAdjacentUnits | GetUnitsInRadius |
|-------|-------------------|------------------|
| **Bridge** | 0 (нет соседей) | 0 (нет радиуса) |
| **3на3** | 8 соседей | Chebyshev distance |
| **Стенка** | ±1 по индексу | |индекс| расстояние |

## 📋 Интерфейс IBattleFormation (9 методов)

```csharp
public interface IBattleFormation
{
    string Name { get; }                                          // Название
    List<IUnit> GetRingUnits(IArmy army);                        // На ринге
    List<IUnit> GetReserveUnits(IArmy army);                     // В запасе
    bool IsOnRing(IUnit unit, IArmy army);                       // Проверка
    IUnit? GetOpponent(IUnit unit, IArmy army, IArmy opponent);  // Противник
    List<IUnit> GetAdjacentUnits(IUnit unit, IArmy army);        // Соседи
    void HandleUnitDeath(IUnit dead, IArmy army, IArmy opp);     // Замена
    void PromoteUnitToRing(IUnit unit, IArmy army);              // Поднять
    (int row, int column) GetUnitPosition(IUnit unit, IArmy);    // Позиция
    List<IUnit> GetUnitsInRadius(IUnit unit, IArmy army, int r); // Рендж
}
```
